# 🏫 Class Management API Guide\n\n## 📚 Available Class APIs\n\n### 🔐 Authentication Required\nSaare APIs ke liye Admin login token chahiye:\n```\nAuthorization: Bearer YOUR_JWT_TOKEN\n```\n\n### 1. **Create Predefined Classes** ⭐ (Recommended)\n```http\nPOST http://localhost:3000/api/admin/classes/predefined\nAuthorization: Bearer YOUR_JWT_TOKEN\n```\n\n**Response:**\n```json\n{\n  \"success\": true,\n  \"message\": \"12 predefined classes created successfully\",\n  \"data\": [...]\n}\n```\n\n**Predefined Classes Include:**\n- 1st A to 12th Science\n- Proper subjects for each grade\n- Room numbers and schedules\n- Appropriate capacity\n\n### 2. **Get All Classes**\n```http\nGET http://localhost:3000/api/admin/classes\nAuthorization: Bearer YOUR_JWT_TOKEN\n```\n\n### 3. **Get Single Class**\n```http\nGET http://localhost:3000/api/admin/classes/CLASS_ID\nAuthorization: Bearer YOUR_JWT_TOKEN\n```\n\n### 4. **Create Custom Class**\n```http\nPOST http://localhost:3000/api/admin/classes\nAuthorization: Bearer YOUR_JWT_TOKEN\nContent-Type: application/json\n\n{\n  \"name\": \"11th Commerce\",\n  \"section\": \"B\",\n  \"grade\": \"11th\",\n  \"subjects\": [\"English\", \"Accountancy\", \"Business Studies\", \"Economics\"],\n  \"capacity\": 35,\n  \"room\": \"603\",\n  \"schedule\": {\n    \"startTime\": \"08:00\",\n    \"endTime\": \"16:00\"\n  }\n}\n```\n\n### 5. **Update Class**\n```http\nPUT http://localhost:3000/api/admin/classes/CLASS_ID\nAuthorization: Bearer YOUR_JWT_TOKEN\nContent-Type: application/json\n\n{\n  \"name\": \"Updated Class Name\",\n  \"capacity\": 40,\n  \"room\": \"New Room Number\"\n}\n```\n\n### 6. **Delete Single Class**\n```http\nDELETE http://localhost:3000/api/admin/classes/CLASS_ID\nAuthorization: Bearer YOUR_JWT_TOKEN\n```\n\n### 7. **Delete All Classes**\n```http\nDELETE http://localhost:3000/api/admin/classes/all\nAuthorization: Bearer YOUR_JWT_TOKEN\n```\n\n## 🚀 Quick Setup Guide\n\n### Step 1: Login as Admin\n```http\nPOST **Classes Already Exist Error**\n- **Problem**: Predefined classes already created\n- **Solution**: Delete all classes first using DELETE /classes/all\n\n### 2. **401 Unauthorized**\n- **Problem**: Token missing या invalid\n- **Solution**: Login करके नया token ले\n\n### 3. **Duplicate Class Error**\n- **Problem**: Same name, section, grade की class already exists\n- **Solution**: Different name/section use करें\n\n## 📝 Testing Checklist\n\n- [ ] Admin login working\n- [ ] Create predefined classes working\n- [ ] Get all classes working\n- [ ] Get single class working\n- [ ] Create
